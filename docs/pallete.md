# Paleta principal

| Uso               | Color           | Hex      |
|-------------------|-----------------|----------|
| Fondo principal   | Marrón oscuro   | #2B1B17  |
| Fondo secundario  | Café madera     | #4A2C24  |
| Color crema       | Beige cálido    | #F2DFC2  |
| Acento            | Terracota       | #B55239  |
| Texto claro       | Marfil          | #FFF7EA  |
| Texto oscuro      | Café intenso    | #1A120F  |

---

## Recomendación de uso

- `#2B1B17` → navbar y footer
- `#F2DFC2` → fondo general
- `#B55239` → botones y hover
- `#FFF7EA` → texto sobre fondos oscuros
- `#4A2C24` → tarjetas o secciones

---

## Estilo que transmite

- Casero
- Gourmet
- Elegante pero acogedor
- Restaurante/cafetería artesanal
